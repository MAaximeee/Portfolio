Projet pedagogique

Contexte

Ce projet est un TP donné à l'école IPSSI. Il s'agit d'une application web permettant à un utilisateur de créer un compte, se connecter, modifier ses informations de profil, rechercher des livres via une API publique (Google Books), et ajouter des livres à ses favoris. L’objectif est de proposer une expérience moderne de gestion de bibliothèque personnelle.

Fonctionnalités principales

  • Inscription et connexion
  • Modification des informations du profil utilisateur
  • Recherche de livres via l’API Google Books
  • Ajout et suppression de livres en favoris
  • Affichage de la liste des livres favoris de l’utilisateur

Technologies utilisées

HTML, CSS, JavaScript, API REST (Google Books), PHP, SQL
