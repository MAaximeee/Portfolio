E6 Event Manager

Contexte

Projet réalisé dans le cadre de l’épreuve E6 du BTS SIO option SLAM, consistant en la conception et le développement d’une plateforme de création et de gestion d’événements. L’application repose sur une architecture complète avec une API développée en Symfony, une interface web en React et une application mobile en React Native. Ce projet m’a permis de mobiliser mes compétences en développement full-stack, notamment sur la gestion des données, la communication entre les différentes couches de l’application et la mise en place de fonctionnalités liées à la gestion des utilisateurs et des événements.

Fonctionnalités principales

  • Création et gestion d’événements
  • Gestion des utilisateurs et des inscriptions
  • Consultation et modification des informations liées aux événements

Technologies utilisées

React JS, React Native, PHP Symfony
