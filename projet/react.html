Projet pedagogique

Contexte

Ce projet pédagogique a marqué mes premiers pas avec React. J’ai profité de cette expérience pour apprendre les bases de ce framework JavaScript moderne et mettre en pratique mes nouvelles compétences à travers plusieurs mini-applications.

Fonctionnalités principales

  • Quiz interactif
  • Recherche de films via une Movie API
  • GeoQuiz avec récupération de données via une API géographique
  • To-do list dynamique
  • Application musicale permettant la recherche d’albums et d’artistes via une API

Technologies utilisées

React, JavaScript, HTML, CSS, APIs externes
